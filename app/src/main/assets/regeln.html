1. Einmal würfeln, um Startspieler zu ermitteln (danach im UZS)
  2. Wenn keine Figur auf dem Spielbrett ist, darf bis zu drei Mal gewürfelt werden (gilt NICHT, wenn im Ziel noch eine Figur bewegt werden könnte)
  3. Bei einer Sechs MUSS eine Figur vom Startfeld ins Spiel gebracht werden. Es sei denn, alle Figuren sind schon im Spiel.
  4. Das Startfeld muss schnellstmöglich von eigenen Figur freigesetzt werden (Es sei denn, es handelt sich um die letzte Figur aus den Startfeldern)
  5. Nach jeder Sechs darf erneut gewürfelt werden
  6. Der Spieler an der Reihe würfelt und setzt eine beliebige seiner Figuren auf dem Feld entsprechend im UZS vor, besetzte Felder werden mitgezählt.
  7. Trifft die Figur mit dem letzten Auge auf ein von einer gegnerischen Figur besetzten Feld, wird diese geschlagen
  8. Es herrscht kein Schlagzwang
  9. Eigene Figuren können nicht geschlagen werden. Gibt es keine andere Möglichkeit, ist der nächste Spieler an der Reihe.
  10. Auf den Zielfeldern dürfen keine Figuren übersprungen werden.
  11. Die letzte Figur darf nur auf dem ersten Zielfeld einrücken
  12. Fremde Zielfelder dürfen nicht betreten werden.
  13. Der Spieler, der zuerst alle Figuren auf den Zielfeldern hat, gewinnt.
  14. Der Rest spielt weiter um die Plätze.
